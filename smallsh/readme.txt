***** Instructions for compiling and executing smallsh.c *****


Step 1: Place the smallsh.c file into your current working directory.

Step 2: Give the smallsh.c file permission to execute by entering the following command: chmod +x smallsh.c

Step 3: Compile smallsh.c with the following command: gcc smallsh.c -o smallsh

Step 4: Execute smallsh with either of the following commands: smallsh  OR  ./smallsh

Step 5: Enter commands into the shell as you wish. Enter "exit" to quit the shell.
